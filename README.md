# Movie Recommendation System

A content-based movie recommendation system built with Python, using cosine similarity to recommend movies based on their features. The system includes a web interface powered by Streamlit and fetches movie posters from The Movie Database (TMDB) API.

## Features

- **Content-Based Recommendations**: Recommends movies similar to the selected one based on genres, cast, crew, and other metadata.
- **Interactive Web App**: User-friendly interface built with Streamlit for easy movie selection and recommendation viewing.
- **Poster Display**: Fetches and displays movie posters directly from TMDB API.
- **Top 5 Recommendations**: Shows the 5 most similar movies for any selected film.

## Installation

1. **Clone or Download the Repository**:
   - Ensure you have the project files in your workspace.

2. **Set Up a Virtual Environment** (Recommended):

   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install Dependencies**:

   ```bash
   pip install -r requirements.txt
   ```

4. **Prepare the Data**:
   - Download the TMDB dataset files:
     - `movies.csv`: Movie metadata from TMDB (available from Kaggle or TMDB API).
     - `credits.csv`: Cast and crew information from TMDB.
   - Place these files in the project directory.
   - Run the Jupyter notebook `movie_recommendation_system.ipynb` to process the data and generate the required pickle files (`movie_dict.pkl` and `similarity.pkl`).

## Usage

1. **Run the Data Processing Notebook**:
   - Open `movie_recommendation_system.ipynb` in Jupyter Notebook or JupyterLab.
   - Execute all cells to generate `movie_dict.pkl` and `similarity.pkl`.

2. **Launch the Web App**:

   ```bash
   streamlit run app.py
   ```

   - Open the provided URL (usually `http://localhost:8501`) in your web browser.

3. **Get Recommendations**:
   - Select a movie from the dropdown or type its name.
   - Click the "Recommend" button to view 5 similar movie recommendations with posters.

## Requirements

- Python 3.7+
- Internet connection (for fetching movie posters via TMDB API)
- TMDB API Key (hardcoded in `app.py` for demo purposes; replace with your own for production)

## Project Structure

- `app.py`: Main Streamlit application for the web interface.
- `movie_recommendation_system.ipynb`: Jupyter notebook for data processing and similarity computation.
- `requirements.txt`: List of Python dependencies.
- `movies.csv`: Raw movie data (not included; download from TMDB or similar source).
- `credits.csv`: Raw credits data (not included; download from TMDB or similar source).
- `movie_dict.pkl`: Pickled movie data (generated by the notebook).
- `similarity.pkl`: Pickled similarity matrix (generated by the notebook).

## Troubleshooting

- **Missing Pickle Files**: Ensure you run the notebook to generate `movie_dict.pkl` and `similarity.pkl`.
- **API Key Issues**: The TMDB API key is hardcoded. For production, obtain your own key from [TMDB](https://www.themoviedb.org/) and update the `fetch_poster` function.
- **Data Files Not Found**: Download `movies.csv` and `credits.csv` from TMDB dataset or similar sources.
- **Streamlit Not Running**: Ensure all dependencies are installed and the virtual environment is activated.
- **Poster Not Loading**: Check your internet connection and API key validity.
